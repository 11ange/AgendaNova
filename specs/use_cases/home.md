# Casos de Uso: Home e Dashboard

## Visualizar Resumo do Dia
**Ator:** Instrutor
**Objetivo:** Ter uma visão rápida dos compromissos imediatos e pendências.
**Funcionalidades:**
- **Agenda do Dia**: Lista das sessões marcadas para a data atual.
- **Pendências Financeiras**: Atalhos ou alertas para sessões realizadas mas com pagamento pendente.
- **Navegação Rápida**: Acesso facilitado para Pacientes, Agenda, Lista de Espera e Relatórios.
